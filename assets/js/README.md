# Quiz & Exam Maker

A web-based Quiz and Exam Generator that creates questions from uploaded PDF or PPTX files.

## Features
- Upload PDF or PPTX
- Quiz mode (practice)
- Exam mode (many items)
- Multiple Choice questions
- Enumeration questions
- Runs fully on the browser
- Deployable via GitHub Pages

## How to Use
1. Upload a PDF or PPTX (text-based)
2. Click "Generate Quiz" or "Generate Exam"
3. Answer the generated questions

## Technologies Used
- HTML, CSS, JavaScript
- PDF.js
- JSZip

## Deployment
This project is designed for GitHub Pages.
